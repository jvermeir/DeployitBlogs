fileCopyExample.readme.txt

This example shows how to use the file plugin to copy a set of files to an overthere host. 

To use:
- copy the file plugin from the available plugins to the plugins directory of your Deployit server installation.
- restart Deployit
- run build.sh: 
	build.sh copyExample
- start the Deployit GUI in a browser, import copyExmple.dar, deploy
